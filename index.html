O que é o Limio?

O Limio é um minerador blockchain construído em C++ moderno, voltado para testes comunitários, segurança e modularidade. Com suporte a testnet e mainnet, oferece mineração descentralizada, modular, transparente e com foco em comunidade.

Como Funciona

Blocos & Transações

Transações são agrupadas em blocos, garantindo segurança e integridade.

Dificuldade Ajustável

Os blocos são minerados com dificuldade configurável para melhor teste e adaptação.

Rede P2P

A rede sincroniza os blocos via P2P, garantindo descentralização.

CLI Intuitiva

Interface de linha de comando para status, mineração e consulta de saldo.

Instalação

Requisitos

  • CMake 3.10+
  • g++/clang++ (C++17)
  • libsodium
  • boost
  • nlohmann_json

Comandos

$ git clone https://github.com/seuusuario/limio.git
$ cd limio
$ ./build.sh
$ ./tests_runner
