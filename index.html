Comece Agora

O que é o Limio?

Limio é um sistema blockchain modular, desenvolvido em C++ moderno.
Focado em simplicidade, transparência, segurança e participação comunitária.
Com o Limio, qualquer pessoa pode minerar, testar, contribuir ou criar sua própria rede.

Como Funciona

Cadeia de Blocos

Cada novo bloco é validado e ligado ao anterior, mantendo histórico e segurança.

Recompensa & Halving

O minerador ganha 10 LIMIO por bloco. A recompensa reduz pela metade a cada 500.000 blocos, até zerar (máximo 64 halvings).

Transações Burn

Permite “queimar” moedas enviando para um endereço especial, removendo LIMIO do sistema.

Tamanho Dinâmico

O bloco cresce conforme a rede, adaptando-se ao uso, de 1 KB até 1 MB.

Segurança & Comunidade

Todos os dados são salvos em arquivos, protegidos e auditáveis. O projeto evolui com ajuda da comunidade.

Diagrama do Fluxo Blockchain

Transações → Blocos → Mineração → Cadeia → Recompensa

Como Instalar

Requisitos

  • CMake 3.10+
  • g++ ou clang++ (C++17)
  • libsodium
  • libboost
  • nlohmann_json

Instalação e Teste

$ git clone https://github.com/seuusuario/limio.git
$ cd limio
$ ./build.sh
$ ./tests_runner

Comandos Simples

Comando Para que serve
./limio status Ver status e conexões
./limio mine Iniciar mineração
./wallet balance <endereço> Consultar saldo
./tests_runner Testar o sistema

Perguntas Frequentes

Como faço para começar a minerar?

Siga os passos na seção de instalação e use o comando ./limio mine para iniciar a mineração.

Como contribuo com o projeto?

Você pode enviar feedback, sugerir melhorias ou contribuir com código via GitHub.

Qual licença do Limio?

O sistema funciona em Windows?

Sim, desde que possua os requisitos instalados.
