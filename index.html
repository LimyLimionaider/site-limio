O que é o Limio?

O Limio é um minerador blockchain construído em C++ moderno, voltado para testes comunitários, segurança e modularidade. Com suporte a testnet e mainnet, oferece mineração descentralizada e configuração simples.

Como Funciona

  • Transações são agrupadas em blocos.
  • Blocos são minerados com dificuldade configurável.
  • A rede sincroniza os blocos via P2P.
  • CLI intuitiva permite status, mineração e consulta de saldo.

Instalação

Requisitos: CMake 3.10+, g++/clang++ (C++17), libsodium, boost, nlohmann_json$ git clone https://github.com/seuusuario/limio.git $ cd limio $ ./build.sh $ ./tests_runner
