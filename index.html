Recursos Avançados

Tecnologia de ponta para o futuro descentralizado

⛏️
Mineração Inteligente

O módulo Miner.cpp executa mineração local otimizada baseada em CPU, com algoritmo leve e eficiente. Validação automática de blocos e integração perfeita à cadeia principal com mecanismo de consenso robusto.

🔗
Rede P2P Avançada

Arquitetura P2PNode.cpp com conexão automática via protocolo handshake. Comunicação autônoma, broadcasting inteligente de blocos e sincronização eficiente entre todos os pares da rede.

🚀
REST API Completa

Sistema RestAPI.cpp e RestServer.cpp oferece endpoints seguros para consultas de carteira, envio de transações, verificação de blocos e acesso completo às métricas da rede.

🗳️
Governança Democrática

Módulo GovernanceQuadratic.cpp implementa votação quadrática, onde o poder é distribuído proporcionalmente ao envolvimento, garantindo decisões justas e participação equilibrada.

Guia para Desenvolvedores

Configure e execute o Limio em minutos

1

Clone o Repositório

Baixe o código fonte do projeto:

git clone https://github.com/LimyLimionaider/limio.git
2

Compile o Projeto

Configure e compile usando CMake:

cd limio && cmake . && make
3

Execute o Sistema

Inicie a blockchain e comece a minerar:

./limio start --mine
